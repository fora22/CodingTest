# Coding_Test CHAPTER 5 DFS/BFS
## DFS
### Depth-First Search
- 깊이 우선 탐색이라고도 부르며, 그래프에서 깊은 부분을 우선적으로 탐색하는 알고리즘이다.
### 동작 원리
- 스택
### 구현 방법
- 재귀 함수 이용
## BFS
### Breadth-First Search
- 너비 우선 탐색이라는 의미를 가지며 가까운 노드부터 탐색하는 알고리즘
### 동작 원리
- 큐
### 구현 방법
- 큐 자료구조 이용
